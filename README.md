# Cancer Multi-Omics Analysis

## Overview
This project analyzes mutation and gene expression data to identify genes affected by mutations.

## Requirements
pandas
matplotlib
seaborn
numpy

## Steps
1. Load mutation data
2. Filter mutations
3. Create mutation matrix
4. Visualize mutations
5. Integrate RNA expression(Given file is not a complete file)
6. Compare mutated vs non-mutated samples

## How to Run
cd scripts
python analysis.py

## Output
- Heatmap of top mutated genes
- Differential expression analysis
